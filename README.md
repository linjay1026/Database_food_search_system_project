# 美食查詢系統

## 專案簡介
本專案為資料庫理論課程的期末專題，主題為「美食查詢系統」。本系統結合前端與後端，讓使用者可以方便地查詢、收藏、評論各地美食餐廳。

## 主要功能
- 餐廳資訊查詢與瀏覽
- 使用者註冊／登入
- 收藏餐廳
- 撰寫與瀏覽餐廳評論
- 圖片上傳與瀏覽

## 系統架構
- 前端：React + Vite
- 後端：Python Flask
- 資料庫：MySQL

## 專案結構

```
2025_DB_final_project
├── /frontend/    前端程式碼
├── /backend/     後端程式碼
├── /sql/         SQL 語法
└── ...
```

## 啟動方式
1. 進入 backend 資料夾，安裝所需套件並啟動 Flask 伺服器：
  ```bash
  cd backend
  pip install -r requirements.txt
  python app.py
  ```
2. 進入 frontend 資料夾，安裝前端套件並啟動前端服務：
  ```bash
  cd frontend
  npm install
  npm run dev
  ```
